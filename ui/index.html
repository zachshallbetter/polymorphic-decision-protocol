Polymorphic Decision Protocol Playground

Open the browser developer console (F12) to inspect and interact with the pdp-core.js library.

The library is bound to window.pdp. You can run operations like:

    pdp.matrixCell(0.8, 0.2) // Returns 'Full'
    pdp.Cents.fromDollars(10.50).toString() // Returns '$10.50'
